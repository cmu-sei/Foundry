Learning Objectives

A learning objective is a statement that specifies measurable behavior that a trainee should exhibit after instruction[1]. Writing learning objectives is an important first step in PCTE training content development. This document recommends a formula long used by military trainers: each learning objective must include one or more tasks, conditions and standards of behavior.

  1. Task: Behavior or action the trainee must exhibit Examples: analyze malware, identify intrusion vector, exploit router
  2. Condition: Situation under which the action will take place Examples: using a Cyber Protection Team (CPT) toolkit, during a distributed denial of service (DDoS) attack, given netflow records and packet capture (PCAP) files
  3. Standard: Metrics specifying satisfactory performance Examples: with 80% accuracy, in 30 minutes, without being observed by blue team

While not mandatory, it is helpful to identify the intended trainees in the learning objective. Here are five examples of PCTE training content learning objectives:

  1. Following this video lesson, the cyber operator will correctly identify 90% of NMap’s options and select which syntax to use given a written scenario.
  2. Following this demonstration video, the cyber operator will identify the components of a Snort IDS rule and select the correct rule to use for specific attacks with 90% accuracy.
  3. Given a standard CPT toolkit connected to an unknown DoD network, the Mission Protect (MP) squad will discover and enumerate 90% of the cyber key terrain within 60 minutes.
  4. Given a security onion system, a packet capture file and IDS logs, the incident response analyst will identify 90% of all malicious IP addresses and downloaded file types within 60 minutes.
  5. Given a standard CPT toolkit, modeled DODIN key terrain and an emulated DoD Industrial Control System (ICS), the Cyber Threat Emulation (CTE) squad will pivot and penetrate the ICS network, compromise the Human Machine Interface (HMI) web server and implant a remote access trojan (RAT) within 60 minutes.

Remember, learning objectives must be clearly stated, achievable and measurable.
